
程式語言:C++
程式語言版本:g++ (Ubuntu 9.3.0-17ubuntu1~20.04) 9.3.0

使用方法:
正常的編譯，並以./a.out執行
程式中有詳細要求每個階段需要做的事
首先是需要輸入樹的order
接下來就是樹的操作，包含以下指令:
Insert      : +<key>,<data>
Delete      : -<key>
Quit        : quit
Show List   : show
Display Tree: display
Help        : help
